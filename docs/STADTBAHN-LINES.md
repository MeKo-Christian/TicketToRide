# Hannover Stadtbahn — Line & Stop Reference

Reference for designing the Ticket to Ride map. Sourced from Wikipedia, ÜSTRA, Moovit/Sporvognsrejser, hannover-verkehr.de, and the MHH transport info page (May 2026). The Stadtbahn has 12 main lines (1–11, 17) running on four tunnel networks.

## Tunnel networks

| Net | Colour | Corridor through the centre | Lines |
|-----|--------|-----------------------------|-------|
| A   | Blue   | Sallstraße ↔ Waterloo ↔ Markthalle/Landtag ↔ Aegi ↔ Kröpcke ↔ Hbf ↔ Sedanstr/Lister Pl. | 3, 7, 9 |
| B   | Red    | Aegi ↔ Kröpcke ↔ Hbf (deepest tunnel, B-Süd to Laatzen/Bothmerstr) | 1, 2, 8 |
| C   | Yellow | Königsworther Platz ↔ Steintor ↔ Kröpcke ↔ Aegi ↔ Marienstraße (eastern fork) | 4, 5, 6, 11 |
| D   | Green  | Surface line through Steintor / Hbf-ZOB / Aegi | 10, 17 |

**Kröpcke** is the central interchange — every Stadtbahn line except 10 and 17 stops there.

## Lines

### Line 1 — Langenhagen ↔ Sarstedt (B, red)

North arm (Langenhagen → Hbf) and south arm (Hbf → Laatzen → Rethen → Sarstedt). Through the centre via B-tunnel: Hbf → Kröpcke → Markthalle/Landtag → Waterloo. South of Waterloo it surfaces and runs via **Bothmerstraße → Döhren → Laatzen → Rethen → Sarstedt** — **does not** go through Allerweg, Linden, or Wettbergen.

Key stops we use: Langenhagen, Vahrenwalder Platz, Hbf, Kröpcke, Aegi (via Markthalle interchange), Waterloo, Rethen, Sarstedt.

### Line 2 — Alte Heide ↔ Rethen (B, red)

North arm to Alte Heide via **Lister Platz**. Through the centre via B-tunnel. South arm shares Line 1's south corridor but turns around at Rethen.

Key stops we use: Alte Heide, Lister Platz, Hbf, Kröpcke, Aegi, Waterloo, Rethen.

### Line 3 — Altwarmbüchen ↔ Wettbergen (A, blue)

North-east arm: Altwarmbüchen → Sahlkamp → Vier Grenzen → Spannhagengarten → … → Hbf. Through the centre via A-tunnel: Hbf → Kröpcke → Aegi → Markthalle → **Waterloo**. South-west arm continues Waterloo → Allerweg → Bothmerstr → **Wettbergen**.

Key stops we use: Altwarmbüchen, Hbf, Kröpcke, Aegi, Markthalle, Waterloo, Allerweg, Wettbergen.

### Line 4 — Garbsen ↔ Roderbruch (C, yellow)

North-west arm: Garbsen → Stöckener Markt → **Stöcken** → Marie-Curie-Allee → … → **Königsworther Platz**. Through centre via C-tunnel: Königsworther Platz → Steintor → Kröpcke → Aegi → Marienstr → Braunschweiger Platz → Geibelstraße → Kantplatz → … → **Medizinische Hochschule** → Roderbruch.

**Important:** the stop near the MHH/Universitätsklinikum is named **"Medizinische Hochschule"**. There is no Stadtbahn stop called "Universität". Leibniz Universität (the campus) is at Königsworther Platz/Welfenplatz.

Key stops we use: Garbsen, Stöcken, Königsworther Platz, Steintor, Kröpcke, Aegi, Medizinische Hochschule, Roderbruch.

### Line 5 — Stöcken ↔ Anderten (C, yellow)

North-west arm: Stöcken → Leinaustraße → … → **Steintor** (NOT via Königsworther Platz — different branch from Line 4). Through centre via C-tunnel: Steintor → Kröpcke → Aegi → Marienstr → Braunschweiger Platz → **Zoo** → Kirchrode → Anderten.

Key stops we use: Stöcken, Steintor, Kröpcke, Aegi, Zoo, Anderten.

### Line 6 — Nordhafen ↔ Messe/Ost (C, yellow)

North arm: Nordhafen → Niedersachsenring → Christuskirche → Goetheplatz → **Steintor**. Through centre via C-tunnel. East/south-east arm: Aegi → Marienstr → … → Bischofshol → … → Messe/Ost. Does **not** pass through Vahrenwalder Platz.

Key stops we use: Nordhafen, Steintor, Kröpcke, Aegi, Messe/Ost.

### Line 7 — Misburg ↔ Wettbergen (A, blue)

North-east arm: Misburg → Forstkamp → Kafkastr → Schierholzstr → Pappelwiese → Paracelsusweg → In den Sieben Stücken → Noltemeyerbrücke → Klingerstr → **Spannhagengarten → Pelikanstr → Vier Grenzen** → … → Hbf. **Does not** pass through Lister Platz. Through centre via A-tunnel. South-west arm: Waterloo → Allerweg → Bothmerstr → **Wettbergen** (shared with Line 3 from Waterloo south).

Key stops we use: Misburg, Hbf, Kröpcke, Aegi, Markthalle, Waterloo, Allerweg, Wettbergen.

### Line 8 — Hbf ↔ Messe/Nord (B, red)

Express to the trade-fair grounds. Centre via B-tunnel: Hbf → Kröpcke → Markthalle → Waterloo. South of Waterloo branches to Messe.

### Line 9 — Fasanenkrug ↔ Empelde (A, blue)

North-east arm: Fasanenkrug → Sahlkamp → Vier Grenzen → Spannhagengarten → … → Hbf (shared NE corridor with Line 3 and Line 7 from Vier Grenzen southwards). Through centre via A-tunnel: Hbf → Kröpcke → Aegi → Markthalle → **Waterloo → Schwarzer Bär → Lindener Marktplatz → Bauweg → Nieschlagstr → Empelde**.

**Important:** to reach **Lindener Marktplatz** from the centre, the line passes **through Waterloo** (and Schwarzer Bär). Lindener Marktplatz is **not** directly connected to Königsworther Platz or Markthalle.

Key stops we use: Fasanenkrug, Hbf, Kröpcke, Aegi, Markthalle, Waterloo, Lindener Marktplatz, Empelde.

### Line 10 — Ahlem ↔ Hbf/Aegi (D, green, mostly surface)

Surface route through the centre. Ahlem → Glocksee → **Königsworther Platz** → Goetheplatz → **Steintor** → Hbf/ZOB → Aegi.

**Important:** **Ahlem connects to Königsworther Platz** (via Glocksee). Ahlem does **not** connect directly to Garbsen — that would mix Line 10 with Line 4.

Key stops we use: Ahlem, Königsworther Platz, Steintor, Hbf, Aegi.

### Line 11 — Haltenhoffstraße ↔ Zoo (C, yellow)

Short north-west arm: Haltenhoffstr → Christuskirche → Goetheplatz → **Steintor** (NOT via Königsworther Platz). Through centre via C-tunnel. East: Steintor → Kröpcke → Aegi → Marienstr → … → Zoo.

Key stops we use: Haltenhoffstraße, Steintor, Kröpcke, Aegi, Zoo.

### Line 17 — Wallensteinstr ↔ Hbf (D, green, surface)

Surface line in the inner city. Connects Waterloo / Markthalle area with the western suburbs via Glocksee. We don't model it explicitly; some routes are labelled `line: 17` as a placeholder.

## Fixes applied to our map

1. **Universität → Medizinische Hochschule** (Line 4). The MHH/Universitätsklinikum stop is the actual name; there's no Stadtbahn stop called "Universität".
2. **Lindener Marktplatz reached via Waterloo** (Line 9: Waterloo → Schwarzer Bär → Linden). Replaces fictional `markthalle-linden` and `koenigsworth-linden`.
3. **Garbsen reached via Stöcken** (Line 4). The fictional `garbsen-ahlem` edge is gone — the two arms only meet at Königsworther Platz.
4. **Haltenhoffstraße → Steintor** (Line 11, Christuskirche/Goetheplatz collapsed). Replaces fictional `haltenhoff-koenigsworth` and `haltenhoff-stoecken`.
5. **Nordhafen reached via Steintor** (Line 6, Christuskirche/Goetheplatz/Niedersachsenring collapsed). Replaces fictional `vahrenwalder-nordhafen` and `stoecken-nordhafen`.
6. **Wettbergen reached via Allerweg** (Lines 3/7, Bothmerstr collapsed). Replaces fictional `linden-wettbergen` and `empelde-wettbergen`.
7. **Roderbruch reached via Medizinische Hochschule** (Line 4 east continuation, Geibelstr/Kantplatz/Misburger Str collapsed). Replaces fictional `zoo-roderbruch`.
8. **Central-south topology**: Markthalle/Landtag is not modelled as a separate node (it collapses into the tunnel corridor). Kröpcke connects straight to Waterloo via `kroepcke-waterloo`, a parallel pair (2 tracks, 2 wagons each). Aegi reaches the south via its own arm (`aegi-rethen`).
9. **Line-attribution corrections**: `lister-bothfeld` is Line 8 (Bothfeld is a Line 8 terminus, not Line 9). `markthalle-waterloo` is Line 1 (shared A/B-tunnel main corridor — Line 17 is the D-surface line and doesn't run this stretch).

## Remaining pragmatic shortcuts (kept for game balance)

These cross-network edges don't correspond to a single real Stadtbahn line, but each one connects two stations whose real corridor passes through intermediate stops we don't model (Spannhagengarten, Vier Grenzen, Sahlkamp, Klingerstr, Pelikanstr). Removing them in isolation would orphan Altwarmbüchen, Fasanenkrug, Misburg, or the Zoo branch from the rest of the graph. We treat them as "feeder/bus" connections:

- `bothfeld-altwarmbuechen` — real path: both reached from Hbf via the NE corridor, but on different branches at Sahlkamp.
- `bothfeld-fasanenkrug` — same NE corridor, different branches.
- `med_hochschule-zoo` — Line 4 and Lines 5/11 fork at Marienstr; not a shared track.
- `messe_ost-anderten` — Line 6 and Line 5 termini, separate SE branches.
- `anderten-misburg` — Line 5 and Line 7 termini; Misburg is reached from Hbf via Pelikanstr/Klingerstr in reality.

If we ever want strict realism we can swap each shortcut for a long collapsed edge from the centre (Hbf-Misburg, Hbf-Altwarmbüchen, Hbf-Fasanenkrug) or introduce the missing NE-corridor stations.
